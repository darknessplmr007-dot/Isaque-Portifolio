Bem-vindo ao meu portfólio pessoal! Este é um site moderno e responsivo que apresenta meus projetos, habilidades e informações de contato. Desenvolvido com foco em design atrativo e interatividade.

## ✨ Funcionalidades

- **Design Dark e Moderno**: Tema preto e vermelho com gradientes e animações suaves.
- **Animações Interativas**: Efeitos de scroll, digitação e hover para uma experiência imersiva.
- **Responsivo**: Otimizado para desktop, tablet e mobile.
- **Projetos Interativos**: Links diretos para demos ao vivo dos projetos.
- **Ícones e Fontes**: Uso de Font Awesome e Google Fonts para um visual profissional.
- **SEO Otimizado**: Metadados e estrutura semântica para melhor indexação.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica e acessível.
- **CSS3**: Estilos avançados com animações, gradientes e responsividade.
- **JavaScript**: Interatividade com ScrollReveal e efeitos customizados.
- **Font Awesome**: Ícones para redes sociais e habilidades.
- **Google Fonts**: Tipografia elegante (DM Sans, Montserrat).

## 🚀 Como Executar

1. **Clone o repositório**:
   ```bash
   git clone https://github.com/darknessplmr007/zack-portifolio.git
   cd zack-portifolio
   ```

2. **Abra no navegador**:
   - Abra o arquivo `index.html` diretamente no seu navegador.
   - Ou use um servidor local (ex.: Live Server no VS Code).

3. **Para desenvolvimento**:
   - Edite os arquivos em `css/style.css` e `js/main.js`.
   - Use um editor como VS Code para live reload.
